HTML

Este lenguaje utiliza marcas para detallar cómo debe aparecer el texto y los gráf i cos en un Navegador web, es decir, se utiliza para estructurar páginas web, por ejemplo, contenidos que podrían ser párrafos, listas con viñetas, imágenes, tablas de datos, videos, animaciones, etc. "Hipertexto" se refiere a enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web.
Los enlaces son un aspecto fundamental de la Web. Al cargar contenido en Internet y vincularlo a páginas creadas por otras personas, usted se convierte en un participante activo en la World Wide Web.

Java Script

Es un lenguaje de programación diseñado para añadir interactividad a las páginas web y crear aplicaciones; es un lenguaje de secuencias de comandos que te permite crear contenido dinámico, por ejemplo, juegos, eventos que ocurren cuando los botones son presionados, formularios, animación, ventanas emergentes y mucho más. Es compatible con todos los navegadores modernos sin necesidad de plug-ins. La mayoría de las páginas web están desarrolladas con las tres tecnologías esenciales en la producción mundial de webs, HTML, CSS y JavaScript.
